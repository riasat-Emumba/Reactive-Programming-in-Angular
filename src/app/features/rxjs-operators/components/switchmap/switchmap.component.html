Switch Map Operators

Map, SwitchAll, and SwitchMap. Map transforms each value emitted by the source observable. SwitchAll and SwitchMap both cancel previous observables and only emit the most recent one. SwitchMap is a combination of map and switchAll, which maps each value to an observable and then flattens all of these inner observables using switchAll.
