CombineLatest & with Latest From

CombineLatest is an RxJS operator that combines multiple observables to create an observable whose values are calculated from the latest values of each of its input observables. Whenever any input observable emits a value, CombineLatest emits the latest values from all input observables.

LatestFrom is another RxJS operator that combines the source observable with another observable to create an observable that emits the latest value from the other observable whenever the source observable emits a value.
