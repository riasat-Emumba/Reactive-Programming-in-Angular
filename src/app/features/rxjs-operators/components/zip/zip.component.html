Zip & ForkJoin

Zip is an RxJS operator that combines multiple observables to create an observable whose values are arrays of the values from each input observable, emitted only when all input observables have emitted at least one value.

ForkJoin is another RxJS operator that combines multiple observables to create an observable that emits an array of the last values emitted by each input observable, only when all input observables have completed.
